Acerca del projecto

Esta web es un practica para la asignatura lenguaje de marcas de 1º de DAW en Florida Universitària.

Para esta práctica he bajado una plantilla sencilla de Bootstrap, la podemos encontrar en https://startbootstrap.com/template/shop-homepage

Alojamiento de la web

Para esta web he utilizado como alojamiento mi propio servidor NAS que dispongo como afición, su sistema operativo es una distribución Linux basada en Debian llamada Open Media Vault.

El proceso es bastante sencillo, como ya tenia instalado el contenedor SWAG para redireccionar de forma segura algunos servicios, he aprobechado su servidor Nginx, simplemente mediante windows utilizando Samba (ficheros compartidos en red) he añadido los componentes de la web a la carpeta www de Nginx.
